## Technology 

Here are the technologies used in this project.

* ReactJs 
* React Native
* NodeJS
* Sass
* Typesript
* NextJs
* react-icons
* axios
* Postgres
* PrismaClient
* Nookies
* JWT


## Getting started

* To install the dependecies.
  - yarn add
  
* To run the project.
  - cd frontend && cd frontend && cd mobile
  - yarn dev
